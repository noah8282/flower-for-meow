# Flower for meow
A simple C program that prints a random love message and grows an ASCII flower.

### Quick Install (One-Liner)
You can download, compile, and install the program globally in just one command. Open your terminal and run:

```bash
curl -sO https://raw.githubusercontent.com/noah8282/flower-for-meow/refs/heads/main/flower.c && gcc -o flower flower.c && sudo mv flower /usr/local/bin/ && rm flower.c
